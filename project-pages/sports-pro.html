SportsPro Technical Support
A customized application that allows employees to track customer incidents/bugs within the SportsPro application. Each incident is then assigned to a technician to work on.
Check out a simulated version of the project: SportPro Technical Support
Project Details / Background
SportsPro is composed of a few different applications that allow customers to keep up to date with their favourite teams. SportsPro Technical Support allows employees connect with customers about technical issues they are facing using SportsPro applications. It allows incidents/bugs to be created, that are connected to the customer, and then allows administrators to assign those incidents to specific technicians. This keeps track of all incidents and makes sure problems are addressed as quickly as possible.
This web application was created using PHP, HTML and CSS, along with a custom database design and MySQL.
